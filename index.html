HTB: Cronos
2020-12-01

Cronos is a Medium Linux Hack The Box machine. This box requires a low level understanding of DNS to enumerate subdomains using a zone transfer. With these new subdomains we gain access to a web terminal through a SQL injection. This gives us reverse shell which we then escalate to root through a cron job. Overall this was a fun box that made you use DNS before you could do anything else.
